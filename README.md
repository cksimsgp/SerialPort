# SerialPort

A simple C# console app for selecting a COM port, setting baud rate, and sending/receiving serial data.

## Run

1. Install the .NET SDK (version 8 or later).
2. From the repository folder:

```bash
dotnet run --project SerialPort.csproj
```

3. Follow the prompts to choose a port and baud rate.
4. Type messages to send. Type `/quit` to exit.
